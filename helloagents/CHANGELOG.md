## [Unreleased]

### 变更
- 按任务书方向将后端认证与权限扩展为 Apache Shiro + RBAC 基础实现，新增验证码、忘记密码、重置密码、会话表与权限表
- 补齐系统管理能力：用户详情/修改、角色权限、系统配置、个人中心、修改密码、仓库管理
- 新增登录日志、操作日志、站内消息、追溯记录、异常单据与预警历史相关后端能力
- 扩展采购模块：采购需求、采购单审核/取消/跟踪/分析、Excel 导入导出
- 扩展销售模块：销售信息发布、销售单审核/取消、应收状态、Excel 导入导出
- 扩展库存模块：真实调拨、盘点盈亏上报、预警记录、Excel 导入导出
- 扩展报表模块：采销存汇总、合规追溯、异常单据、联动分析与 Excel 导出
- 补充面向任务书流程的后端集成测试样例
- 新增前端第二轮优化：引入 TypeScript 核心基础设施、`vue-tsc` 类型校验与无权限提示页
- 优化前端构建拆包与图表懒加载，降低主入口包体压力
- 增强按钮级权限控制、登录错误提示、驾驶舱加载/空态与表格加载态
- 重构前端页面为 Element Plus 后台工作台风格，新增登录页、驾驶舱、个人中心与暗黑模式
- 新增动态路由与权限菜单机制，按角色加载采购、销售、库存、管理员和统计追溯页面
- 新增统一表格/图表/导出组件，补齐采购、销售、库存、管理员、经营大屏与合规追溯页面
- 修正文档与代码现状不一致的问题，更新 README、项目技术约定与知识库
- 将 `/api/reports/export` 从 CSV 扩展为 Excel 导出能力，前端仍需按后端真实输出联调
- 修复报表中心和商品列表的前后端字段/接口使用，统一以后端真实返回为准
- 将采购流程拆分为“到货”和“入库”两个动作，新增 `RECEIVED` 状态与到货时间
- 将首页概览统计调整为数据库实时统计值，移除硬编码数据

## [0.1.0] - 2026-03-23

- 初始化烟草采销存协同管理平台前后端分离工程
- 新增 Spring Boot 后端骨架与健康检查/概览接口
- 新增 Vue 3 + Vite 前端骨架与核心业务页面导航
- 新增 HelloAGENTS 知识库、方案包与历史索引
- 新增采购、库存、销售、报表模块的前后端实现与临时 MySQL 数据闭环
- 新增报表 CSV 导出与 ECharts 趋势图页面
