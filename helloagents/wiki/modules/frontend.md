# frontend

## 目的
提供 Vue 单页应用工作台，承载登录、驾驶舱、系统管理、采购、销售、库存、报表和追溯页面。

## 模块概述
- **职责:** 登录鉴权、动态路由、页面级 / 按钮级权限控制、后台布局、图表展示、导出交互、暗黑模式、消息提醒与 TypeScript 基础设施
- **状态:** ✅可用
- **最后更新:** 2026-03-30

## 标准启动命令
- 构建：`cd frontend && npm install && npm run build`
- 运行：`cd backend && mvn spring-boot:run`
- 调试：`cd frontend && npm run dev`

## 规范
- 默认交付方式为后端静态托管，构建产物写入 `backend/src/main/resources/static/`
- API 统一走 `src/api/*`，基础地址固定为 `/api`
- 路由以 `src/router/route-map.ts` 为准，菜单分组由 `src/utils/access.ts` 构建
- 登录态、权限、消息、主题状态统一由 Pinia 管理
- 构建前必须通过 `npm run type-check`
- 页面字段、状态流转与权限显示以当前后端真实返回为准

## 当前实现
- 登录、忘记密码、重置密码已接入真实后端接口
- 工作台包含驾驶舱、个人中心、消息中心与暗黑模式开关
- 管理端覆盖账号、角色权限、基础资料、日志、系统配置、仓库维护
- 采购、销售、库存页面均支持真实业务动作与权限控制
- 报表支持经营大屏、联动分析、异常审核与 Excel 导出

## 最新变更
- 驾驶舱新增“历史烟品销售对比”图表，支持数量 / 金额维度切换
- 采购入库、销售出库、库存调拨、库存盘点均已改为按仓库执行

## 注意事项

- 常规演示请访问后端端口，不要长期依赖 `5173`
