# 变更提案: 前端页面重构与后台工作台升级

## 需求背景
用户已提供完整的 Vue 3 前端页面规划，希望将当前较为简化的前端改造成更接近企业后台系统的页面体系，覆盖登录、布局、工作台、采购、销售、库存、超级管理员、统计追溯等页面，并要求支持权限控制、统一表格与图表风格、深绿+金色行业主题以及更完整的交互体验。

## 变更目标
1. 将当前简单列表型前端重构为基于 Element Plus 的后台工作台风格界面。
2. 引入动态菜单、权限守卫、个人中心、消息提醒、暗黑模式等通用能力。
3. 按模块补齐采购、销售、库存、超级管理员、统计追溯等页面骨架与交互。
4. 保持与现有后端 API 的兼容，优先复用已有接口完成真实数据展示。
5. 同步更新知识库与变更记录，确保架构与页面现状一致。

## 影响范围
- **模块:** frontend, admin, procurement, sales, inventory, report, helloagents/wiki
- **文件:** `frontend/src/*`、`frontend/package.json`、知识库文档、变更历史索引
- **交互:** 登录页、主布局、动态路由、页面导航、导出、图表与消息通知

## 成功标准
- 前端成功切换到新的后台布局与主题风格。
- 路由支持按角色/权限过滤菜单与页面访问。
- 采购、销售、库存、管理、统计追溯页面均可访问并具备完整 UI 骨架。
- `npm run build` 构建通过。
- 知识库、Changelog、历史索引与代码现状保持一致。
